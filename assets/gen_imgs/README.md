# Generated Character Images

This directory contains character images generated by `scripts/generate_character_images.py`.

Images are named in the format: `{expression}_{action}.jpg`

## Expressions
- neutral   # 普通 / 默认
- smile     # 微笑
- serious   # 严肃
- confused  # 困惑
- surprised # 惊讶
- sad       # 低落

## Actions
- none      # 无动作
- nod       # 点头
- shake     # 摇头
- wave      # 挥手
- jump      # 跳跃
- point     # 指向

Run `uv run python scripts/generate_character_images.py` to generate all combinations.
